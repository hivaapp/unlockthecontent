## Part 4 — Cloudflare R2 Setup + File Upload Service

---

### What This Part Does

Creates three R2 buckets with correct CORS and access policies. Writes a Supabase Edge Function that generates presigned upload URLs so the browser uploads directly to R2 — never through your server. Writes a second Edge Function that generates presigned download URLs for serving locked content securely. Builds the complete frontend file upload service with progress tracking, type validation, size limits, and retry logic. Connects file metadata to the `files` table in Supabase. By the end of this part every file upload in the app — content files, sponsor videos, completion assets, chat attachments — flows through this single service.

---

### Architecture Decision — Why Presigned URLs

The browser never talks to Supabase to upload files. The flow is:

```
Browser → Supabase Edge Function → R2 presigned URL → Browser uploads directly to R2
                                                     → Browser tells Supabase "upload done"
                                                     → Supabase saves file metadata to files table
```

This keeps egress costs near zero. The file goes browser → R2 directly. Supabase only handles the small metadata record. R2 egress to the internet is free. Supabase storage egress is not — which is why you chose R2.

### Block 5 — R2 Signing Utility (Shared by Edge Functions)

```typescript
// supabase/functions/_shared/r2.ts

const R2_ACCOUNT_ID = Deno.env.get('R2_ACCOUNT_ID')!
const R2_ACCESS_KEY_ID = Deno.env.get('R2_ACCESS_KEY_ID')!
const R2_SECRET_ACCESS_KEY = Deno.env.get('R2_SECRET_ACCESS_KEY')!
const R2_ENDPOINT = Deno.env.get('R2_ENDPOINT')!

// Bucket names
export const BUCKETS = {
  content: Deno.env.get('R2_BUCKET_CONTENT')!,
  sponsor: Deno.env.get('R2_BUCKET_SPONSOR')!,
  assets:  Deno.env.get('R2_BUCKET_ASSETS')!,
} as const

export type BucketName = keyof typeof BUCKETS

// ── AWS Signature V4 helpers ─────────────────────────────────────────────

async function sha256(message: string): Promise<ArrayBuffer> {
  const encoder = new TextEncoder()
  return await crypto.subtle.digest('SHA-256', encoder.encode(message))
}

function toHex(buffer: ArrayBuffer): string {
  return Array.from(new Uint8Array(buffer))
    .map(b => b.toString(16).padStart(2, '0'))
    .join('')
}

async function hmac(key: ArrayBuffer | string, message: string): Promise<ArrayBuffer> {
  const encoder = new TextEncoder()
  const keyData = typeof key === 'string' ? encoder.encode(key) : key
  const cryptoKey = await crypto.subtle.importKey(
    'raw', keyData, { name: 'HMAC', hash: 'SHA-256' }, false, ['sign']
  )
  return await crypto.subtle.sign('HMAC', cryptoKey, encoder.encode(message))
}

function formatDate(date: Date): string {
  return date.toISOString().replace(/[-:]/g, '').replace(/\.\d{3}/, '')
}

function formatDateShort(date: Date): string {
  return formatDate(date).substring(0, 8)
}

// ── Generate presigned PUT URL (for upload) ──────────────────────────────

export async function generatePresignedPutUrl(params: {
  bucket: string
  key: string
  contentType: string
  expiresInSeconds?: number
  maxFileSizeBytes?: number
}): Promise<string> {
  const {
    bucket,
    key,
    contentType,
    expiresInSeconds = 3600,   // 1 hour default
    maxFileSizeBytes,
  } = params

  const now = new Date()
  const dateShort = formatDateShort(now)
  const datetime = formatDate(now)
  const region = 'auto'
  const service = 's3'

  const host = `${R2_ACCOUNT_ID}.r2.cloudflarestorage.com`
  const url = new URL(`https://${host}/${bucket}/${encodeURIComponent(key)}`)

  const credential = `${R2_ACCESS_KEY_ID}/${dateShort}/${region}/${service}/aws4_request`

  const queryParams: Record<string, string> = {
    'X-Amz-Algorithm':     'AWS4-HMAC-SHA256',
    'X-Amz-Credential':    credential,
    'X-Amz-Date':          datetime,
    'X-Amz-Expires':       String(expiresInSeconds),
    'X-Amz-SignedHeaders': 'content-type;host',
  }

  if (maxFileSizeBytes) {
    queryParams['x-amz-meta-max-size'] = String(maxFileSizeBytes)
  }

  // Sort query params alphabetically for canonical request
  const sortedParams = Object.keys(queryParams)
    .sort()
    .map(k => `${encodeURIComponent(k)}=${encodeURIComponent(queryParams[k])}`)
    .join('&')

  url.search = sortedParams

  // Canonical request
  const canonicalRequest = [
    'PUT',
    `/${bucket}/${encodeURIComponent(key)}`,
    sortedParams,
    `content-type:${contentType}\nhost:${host}\n`,
    'content-type;host',
    'UNSIGNED-PAYLOAD',
  ].join('\n')

  // String to sign
  const hashedCanonical = toHex(await sha256(canonicalRequest))
  const stringToSign = [
    'AWS4-HMAC-SHA256',
    datetime,
    `${dateShort}/${region}/${service}/aws4_request`,
    hashedCanonical,
  ].join('\n')

  // Signing key
  const dateKey      = await hmac(`AWS4${R2_SECRET_ACCESS_KEY}`, dateShort)
  const regionKey    = await hmac(dateKey, region)
  const serviceKey   = await hmac(regionKey, service)
  const signingKey   = await hmac(serviceKey, 'aws4_request')
  const signature    = toHex(await hmac(signingKey, stringToSign))

  url.searchParams.set('X-Amz-Signature', signature)

  return url.toString()
}

// ── Delete an object from R2 ─────────────────────────────────────────────

export async function deleteR2Object(bucket: string, key: string): Promise<boolean> {
  const now = new Date()
  const dateShort = formatDateShort(now)
  const datetime = formatDate(now)
  const region = 'auto'
  const service = 's3'

  const host = `${R2_ACCOUNT_ID}.r2.cloudflarestorage.com`
  const credential = `${R2_ACCESS_KEY_ID}/${dateShort}/${region}/${service}/aws4_request`

  const canonicalRequest = [
    'DELETE',
    `/${bucket}/${encodeURIComponent(key)}`,
    '',
    `host:${host}\nx-amz-date:${datetime}\n`,
    'host;x-amz-date',
    toHex(await sha256('')),
  ].join('\n')

  const hashedCanonical = toHex(await sha256(canonicalRequest))
  const stringToSign = [
    'AWS4-HMAC-SHA256',
    datetime,
    `${dateShort}/${region}/${service}/aws4_request`,
    hashedCanonical,
  ].join('\n')

  const dateKey    = await hmac(`AWS4${R2_SECRET_ACCESS_KEY}`, dateShort)
  const regionKey  = await hmac(dateKey, region)
  const serviceKey = await hmac(regionKey, service)
  const signingKey = await hmac(serviceKey, 'aws4_request')
  const signature  = toHex(await hmac(signingKey, stringToSign))

  const response = await fetch(
    `https://${host}/${bucket}/${encodeURIComponent(key)}`,
    {
      method: 'DELETE',
      headers: {
        'Host': host,
        'x-amz-date': datetime,
        'Authorization': `AWS4-HMAC-SHA256 Credential=${credential}, SignedHeaders=host;x-amz-date, Signature=${signature}`,
      },
    }
  )

  return response.status === 204 || response.status === 200
}
```

### Block 6 — Edge Function: `generate-upload-url`

Create `supabase/functions/generate-upload-url/index.ts`.

This function is called by the frontend before every file upload. It validates the request, generates the presigned PUT URL, and returns it along with the R2 key and file record ID that the frontend will use.

```typescript
// supabase/functions/generate-upload-url/index.ts
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import { generatePresignedPutUrl, BUCKETS, BucketName } from '../_shared/r2.ts'

// ── File type and size rules ─────────────────────────────────────────────

const FILE_RULES = {
  content: {
    // PDFs, docs, images, archives — anything a creator locks
    allowedMimeTypes: [
      'application/pdf',
      'image/jpeg', 'image/png', 'image/gif', 'image/webp',
      'video/mp4', 'video/quicktime', 'video/webm',
      'application/zip', 'application/x-zip-compressed',
      'application/vnd.openxmlformats-officedocument.presentationml.presentation',
      'application/vnd.openxmlformats-officedocument.wordprocessingml.document',
      'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet',
      'text/plain', 'text/csv',
      'application/json',
      'application/octet-stream', // catch-all for unknown
    ],
    maxSizeBytes: 500 * 1024 * 1024, // 500 MB
  },
  sponsor: {
    // Sponsor videos only
    allowedMimeTypes: [
      'video/mp4', 'video/quicktime', 'video/webm', 'video/avi',
    ],
    maxSizeBytes: 500 * 1024 * 1024, // 500 MB
  },
  assets: {
    // Completion assets, chat attachments, profile things
    allowedMimeTypes: [
      'application/pdf',
      'image/jpeg', 'image/png', 'image/gif', 'image/webp',
      'video/mp4', 'video/quicktime', 'video/webm',
      'application/zip',
      'application/vnd.openxmlformats-officedocument.presentationml.presentation',
      'application/vnd.openxmlformats-officedocument.wordprocessingml.document',
      'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet',
      'text/plain',
    ],
    maxSizeBytes: 100 * 1024 * 1024, // 100 MB
  },
}

// ── Derive file_type enum value from MIME ────────────────────────────────

function getFileType(mimeType: string): string {
  if (mimeType === 'application/pdf') return 'pdf'
  if (mimeType.startsWith('image/')) return 'image'
  if (mimeType.startsWith('video/')) return 'video'
  if (mimeType.includes('spreadsheet') || mimeType === 'text/csv') return 'spreadsheet'
  if (mimeType.includes('word') || mimeType === 'text/plain') return 'document'
  if (mimeType.includes('presentation')) return 'document'
  if (mimeType.includes('zip')) return 'archive'
  return 'other'
}

// ── Generate R2 object key ───────────────────────────────────────────────

function generateR2Key(userId: string, bucket: BucketName, fileName: string): string {
  const timestamp = Date.now()
  const random = Math.random().toString(36).substring(2, 10)
  const ext = fileName.includes('.') ? '.' + fileName.split('.').pop()!.toLowerCase() : ''
  // Structure: userId/year-month/timestamp-random.ext
  // This gives natural partitioning and avoids hotspotting
  const date = new Date()
  const yearMonth = `${date.getFullYear()}-${String(date.getMonth() + 1).padStart(2, '0')}`
  return `${userId}/${yearMonth}/${timestamp}-${random}${ext}`
}

// ── Main handler ─────────────────────────────────────────────────────────

serve(async (req: Request) => {
  // Handle CORS preflight
  if (req.method === 'OPTIONS') {
    return new Response(null, {
      status: 204,
      headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'POST, OPTIONS',
        'Access-Control-Allow-Headers': 'Content-Type, Authorization',
      },
    })
  }

  if (req.method !== 'POST') {
    return Response.json({ error: 'Method not allowed' }, { status: 405 })
  }

  // ── Auth check ───────────────────────────────────────────────────────
  const authHeader = req.headers.get('Authorization')
  if (!authHeader) {
    return Response.json({ error: 'Missing authorization header' }, { status: 401 })
  }

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!,
  )

  // Verify the user's JWT
  const token = authHeader.replace('Bearer ', '')
  const { data: { user }, error: authError } = await supabase.auth.getUser(token)

  if (authError || !user) {
    return Response.json({ error: 'Invalid or expired token' }, { status: 401 })
  }

  // ── Parse request body ───────────────────────────────────────────────
  let body: {
    fileName: string
    mimeType: string
    sizeBytes: number
    bucket: BucketName
    // Optional: link the file to a specific purpose for metadata
    purpose?: 'content' | 'sponsor_video' | 'completion_asset' | 'chat_attachment' | 'broadcast_image'
  }

  try {
    body = await req.json()
  } catch {
    return Response.json({ error: 'Invalid JSON body' }, { status: 400 })
  }

  const { fileName, mimeType, sizeBytes, bucket, purpose } = body

  // ── Validate bucket ──────────────────────────────────────────────────
  if (!BUCKETS[bucket]) {
    return Response.json({ error: `Invalid bucket. Must be one of: ${Object.keys(BUCKETS).join(', ')}` }, { status: 400 })
  }

  // ── Validate file type ───────────────────────────────────────────────
  const rules = FILE_RULES[bucket]
  if (!rules.allowedMimeTypes.includes(mimeType)) {
    return Response.json({
      error: `File type not allowed for this bucket.`,
      allowedTypes: rules.allowedMimeTypes,
    }, { status: 400 })
  }

  // ── Validate file size ───────────────────────────────────────────────
  if (sizeBytes > rules.maxSizeBytes) {
    return Response.json({
      error: `File too large. Maximum size is ${rules.maxSizeBytes / (1024 * 1024)}MB.`,
      maxSizeBytes: rules.maxSizeBytes,
    }, { status: 400 })
  }

  if (sizeBytes <= 0) {
    return Response.json({ error: 'File size must be greater than 0.' }, { status: 400 })
  }

  // ── Generate R2 key ──────────────────────────────────────────────────
  const r2Key = generateR2Key(user.id, bucket, fileName)
  const r2Bucket = BUCKETS[bucket]

  // ── Create file record in DB (pending upload) ────────────────────────
  const { data: fileRecord, error: dbError } = await supabase
    .from('files')
    .insert({
      owner_id: user.id,
      r2_key: r2Key,
      r2_bucket: r2Bucket,
      original_name: fileName,
      file_type: getFileType(mimeType),
      mime_type: mimeType,
      size_bytes: sizeBytes,
      is_placeholder: true, // marked true until upload confirms
    })
    .select('id')
    .single()

  if (dbError) {
    console.error('DB insert error:', dbError)
    return Response.json({ error: 'Failed to create file record.' }, { status: 500 })
  }

  // ── Generate presigned PUT URL ───────────────────────────────────────
  const presignedUrl = await generatePresignedPutUrl({
    bucket: r2Bucket,
    key: r2Key,
    contentType: mimeType,
    expiresInSeconds: 3600, // URL valid for 1 hour
    maxFileSizeBytes: sizeBytes,
  })

  return Response.json({
    fileId: fileRecord.id,
    r2Key,
    presignedUrl,
    expiresAt: new Date(Date.now() + 3600 * 1000).toISOString(),
  }, {
    headers: { 'Access-Control-Allow-Origin': '*' },
  })
})
```

### Block 7 — Edge Function: `confirm-upload`

After the browser uploads to R2 successfully it calls this function to confirm the upload. This marks `is_placeholder = false` in the DB — the file is now live.

```typescript
// supabase/functions/confirm-upload/index.ts
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'

serve(async (req: Request) => {
  if (req.method === 'OPTIONS') {
    return new Response(null, {
      status: 204,
      headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'POST, OPTIONS',
        'Access-Control-Allow-Headers': 'Content-Type, Authorization',
      },
    })
  }

  const authHeader = req.headers.get('Authorization')
  if (!authHeader) {
    return Response.json({ error: 'Unauthorized' }, { status: 401 })
  }

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!,
  )

  const token = authHeader.replace('Bearer ', '')
  const { data: { user }, error: authError } = await supabase.auth.getUser(token)
  if (authError || !user) {
    return Response.json({ error: 'Invalid token' }, { status: 401 })
  }

  const { fileId, durationSeconds, widthPx, heightPx } = await req.json()

  if (!fileId) {
    return Response.json({ error: 'fileId is required' }, { status: 400 })
  }

  // Verify this file belongs to the authenticated user
  const { data: file } = await supabase
    .from('files')
    .select('id, owner_id, is_placeholder')
    .eq('id', fileId)
    .single()

  if (!file) {
    return Response.json({ error: 'File not found' }, { status: 404 })
  }

  if (file.owner_id !== user.id) {
    return Response.json({ error: 'Forbidden' }, { status: 403 })
  }

  if (!file.is_placeholder) {
    // Already confirmed — idempotent, just return success
    return Response.json({ success: true, fileId })
  }

  // Mark as confirmed and update optional metadata
  const updates: Record<string, unknown> = { is_placeholder: false }
  if (durationSeconds) updates.duration_seconds = durationSeconds
  if (widthPx) updates.width_px = widthPx
  if (heightPx) updates.height_px = heightPx

  const { error: updateError } = await supabase
    .from('files')
    .update(updates)
    .eq('id', fileId)

  if (updateError) {
    return Response.json({ error: 'Failed to confirm upload' }, { status: 500 })
  }

  return Response.json({ success: true, fileId }, {
    headers: { 'Access-Control-Allow-Origin': '*' },
  })
})
```

### Block 8 — Edge Function: `get-download-url`

Called whenever locked content needs to be served. Validates that the viewer has completed the unlock action before generating the presigned download URL.

```typescript
// supabase/functions/get-download-url/index.ts
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import { generatePresignedGetUrl, BUCKETS } from '../_shared/r2.ts'

serve(async (req: Request) => {
  if (req.method === 'OPTIONS') {
    return new Response(null, {
      status: 204,
      headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'POST, OPTIONS',
        'Access-Control-Allow-Headers': 'Content-Type, Authorization',
      },
    })
  }

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!,
  )

  const {
    fileId,
    linkSlug,
    unlockType,
    sessionKey,       // for anonymous viewers
    forceDownload,    // true = attachment, false = inline (for video streaming)
  } = await req.json()

  if (!fileId || !linkSlug || !unlockType) {
    return Response.json({ error: 'fileId, linkSlug, and unlockType are required' }, { status: 400 })
  }

  // ── Get file record ──────────────────────────────────────────────────
  const { data: file } = await supabase
    .from('files')
    .select('id, r2_key, r2_bucket, original_name, mime_type, is_placeholder')
    .eq('id', fileId)
    .single()

  if (!file || file.is_placeholder) {
    return Response.json({ error: 'File not found or not yet uploaded' }, { status: 404 })
  }

  // ── Get authenticated user (optional — anon viewers are allowed) ─────
  const authHeader = req.headers.get('Authorization')
  let userId: string | null = null

  if (authHeader) {
    const token = authHeader.replace('Bearer ', '')
    const { data: { user } } = await supabase.auth.getUser(token)
    userId = user?.id ?? null
  }

  // ── Verify unlock completion ─────────────────────────────────────────
  // Each unlock type has different verification logic.
  // This prevents serving content to viewers who haven't completed the unlock.
  const { data: link } = await supabase
    .from('links')
    .select('id, is_active')
    .eq('slug', linkSlug)
    .single()

  if (!link || !link.is_active) {
    return Response.json({ error: 'Link not found or inactive' }, { status: 404 })
  }

  let isVerified = false

  if (unlockType === 'email_subscribe') {
    // Verify via sessionKey stored in email_subscribers
    const { data: subscriber } = await supabase
      .from('email_subscribers')
      .select('id')
      .eq('link_id', link.id)
      .eq('session_key', sessionKey)
      .single()
    isVerified = !!subscriber

  } else if (unlockType === 'social_follow') {
    // Verify all follows completed
    const query = supabase
      .from('social_unlocks')
      .select('id, all_completed')
      .eq('link_id', link.id)
      .eq('all_completed', true)

    if (userId) query.eq('viewer_id', userId)
    else query.eq('session_key', sessionKey)

    const { data: unlock } = await query.single()
    isVerified = !!unlock

  } else if (unlockType === 'custom_sponsor') {
    // Verify video was watched
    const query = supabase
      .from('sponsor_impressions')
      .select('id, watch_completed')
      .eq('link_id', link.id)
      .eq('watch_completed', true)

    if (userId) query.eq('viewer_id', userId)
    else query.eq('session_key', sessionKey)

    const { data: impression } = await query.single()
    isVerified = !!impression

  } else if (unlockType === 'completion_asset') {
    // Verify pairing session is completed and user is a participant
    if (!userId) {
      return Response.json({ error: 'Authentication required for completion assets' }, { status: 401 })
    }
    const isCompleted = await supabase.rpc('user_has_completed_session', {
      p_user_id: userId,
      p_link_id: link.id,
    })
    isVerified = isCompleted.data === true

  } else if (unlockType === 'owner') {
    // Creator accessing their own file
    const { data: linkOwner } = await supabase
      .from('links')
      .select('creator_id')
      .eq('id', link.id)
      .single()
    isVerified = linkOwner?.creator_id === userId
  }

  if (!isVerified) {
    return Response.json({
      error: 'Unlock not verified. Complete the required action first.',
    }, { status: 403 })
  }

  // ── Generate presigned GET URL ───────────────────────────────────────
  const contentDisposition = forceDownload
    ? `attachment; filename="${encodeURIComponent(file.original_name)}"`
    : 'inline'

  const downloadUrl = await generatePresignedGetUrl({
    bucket: file.r2_bucket,
    key: file.r2_key,
    expiresInSeconds: 300, // 5 minutes — short because it's post-unlock
    responseContentDisposition: contentDisposition,
    responseContentType: file.mime_type,
  })

  return Response.json({ downloadUrl, expiresIn: 300 }, {
    headers: { 'Access-Control-Allow-Origin': '*' },
  })
})
```

### Block 9 — Edge Function: `delete-file`

Called when a creator deletes a link or removes a file. Deletes from R2 then removes the DB record.

```typescript
// supabase/functions/delete-file/index.ts
import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2'
import { deleteR2Object } from '../_shared/r2.ts'

serve(async (req: Request) => {
  if (req.method === 'OPTIONS') {
    return new Response(null, {
      status: 204,
      headers: {
        'Access-Control-Allow-Origin': '*',
        'Access-Control-Allow-Methods': 'POST, OPTIONS',
        'Access-Control-Allow-Headers': 'Content-Type, Authorization',
      },
    })
  }

  const authHeader = req.headers.get('Authorization')
  if (!authHeader) return Response.json({ error: 'Unauthorized' }, { status: 401 })

  const supabase = createClient(
    Deno.env.get('SUPABASE_URL')!,
    Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!,
  )

  const token = authHeader.replace('Bearer ', '')
  const { data: { user }, error: authError } = await supabase.auth.getUser(token)
  if (authError || !user) return Response.json({ error: 'Invalid token' }, { status: 401 })

  const { fileId } = await req.json()
  if (!fileId) return Response.json({ error: 'fileId required' }, { status: 400 })

  // Verify ownership
  const { data: file } = await supabase
    .from('files')
    .select('id, owner_id, r2_key, r2_bucket')
    .eq('id', fileId)
    .single()

  if (!file) return Response.json({ error: 'File not found' }, { status: 404 })
  if (file.owner_id !== user.id) return Response.json({ error: 'Forbidden' }, { status: 403 })

  // Delete from R2 first
  const r2Deleted = await deleteR2Object(file.r2_bucket, file.r2_key)

  if (!r2Deleted) {
    console.error(`R2 delete failed for key: ${file.r2_key}`)
    // Continue anyway — orphaned R2 objects are less bad than orphaned DB records
  }

  // Delete DB record (cascades to links via ON DELETE SET NULL)
  const { error: dbError } = await supabase
    .from('files')
    .delete()
    .eq('id', fileId)

  if (dbError) {
    return Response.json({ error: 'Failed to delete file record' }, { status: 500 })
  }

  return Response.json({ success: true }, {
    headers: { 'Access-Control-Allow-Origin': '*' },
  })
})
```

```javascript
const EDGE_FUNCTION_BASE = import.meta.env.VITE_SUPABASE_URL + '/functions/v1'

// ── File validation rules (mirrors server-side rules) ────────────────────

    ],
  },
}

// ── Client-side validation ────────────────────────────────────────────────

export const validateFile = (file, bucket = 'content') => {
  const rules = FILE_RULES[bucket]
  if (!rules) return { valid: false, error: 'Invalid bucket type.' }

  if (file.size > rules.maxSizeBytes) {
    return {
      valid: false,
      error: `File is too large. Maximum size is ${rules.maxSizeMB}MB. Your file is ${(file.size / (1024 * 1024)).toFixed(1)}MB.`,
    }
  }

  if (file.size === 0) {
    return { valid: false, error: 'File is empty.' }
  }

  const ext = file.name.split('.').pop()?.toLowerCase()
  if (!rules.allowedExtensions.includes(ext)) {
    return {
      valid: false,
      error: `File type .${ext} is not supported. Allowed types: ${rules.allowedExtensions.join(', ')}.`,
    }
  }

  return { valid: true, error: null }
}

// ── Format file size for display ─────────────────────────────────────────

export const formatFileSize = (bytes) => {
  if (bytes === 0) return '0 B'
  const k = 1024
  const sizes = ['B', 'KB', 'MB', 'GB']
  const i = Math.floor(Math.log(bytes) / Math.log(k))
  return `${parseFloat((bytes / Math.pow(k, i)).toFixed(1))} ${sizes[i]}`
}

// ── Get file emoji for display ────────────────────────────────────────────

export const getFileEmoji = (fileName, mimeType = '') => {
  const ext = fileName?.split('.').pop()?.toLowerCase()
  if (ext === 'pdf' || mimeType === 'application/pdf') return '📄'
  if (['jpg', 'jpeg', 'png', 'gif', 'webp'].includes(ext) || mimeType.startsWith('image/')) return '🖼️'
  if (['mp4', 'mov', 'webm', 'avi'].includes(ext) || mimeType.startsWith('video/')) return '🎬'
  if (['zip', 'rar', '7z'].includes(ext)) return '📦'
  if (['pptx', 'ppt'].includes(ext)) return '📊'
  if (['docx', 'doc'].includes(ext)) return '📝'
  if (['xlsx', 'csv'].includes(ext)) return '📈'
  if (['txt', 'md'].includes(ext)) return '📋'
  if (['json', 'js', 'ts', 'py', 'css', 'html'].includes(ext)) return '💾'
  return '📎'
}

// ── Get auth headers ──────────────────────────────────────────────────────

const getAuthHeaders = async () => {
  const { data: { session } } = await supabase.auth.getSession()
  if (!session?.access_token) throw new Error('Not authenticated')
  return {
    'Authorization': `Bearer ${session.access_token}`,
    'Content-Type': 'application/json',
  }
}

// ── Main upload function ──────────────────────────────────────────────────
//
// Usage:
//   const result = await uploadFile(file, 'content', {
//     onProgress: (percent) => setProgress(percent),
//     onStageChange: (stage) => setStage(stage),
//   })
//   // result = { fileId, r2Key, originalName, mimeType, sizeBytes }

export const uploadFile = async (file, bucket = 'content', options = {}) => {
  const {
    onProgress = () => {},
    onStageChange = () => {},
    signal, // AbortController signal for cancellation
  } = options

  // ── Step 1: Client-side validation ──────────────────────────────────
  onStageChange('validating')
  const validation = validateFile(file, bucket)
  if (!validation.valid) throw new Error(validation.error)

  // ── Step 2: Get presigned URL from Edge Function ─────────────────────
  onStageChange('preparing')
  onProgress(5)

  const headers = await getAuthHeaders()

  const urlResponse = await fetch(`${EDGE_FUNCTION_BASE}/generate-upload-url`, {
    method: 'POST',
    headers,
    signal,
    body: JSON.stringify({
      fileName: file.name,
      mimeType: file.type || 'application/octet-stream',
      sizeBytes: file.size,
      bucket,
    }),
  })

  if (!urlResponse.ok) {
    const err = await urlResponse.json()
    throw new Error(err.error || 'Failed to prepare upload.')
  }

  const { fileId, r2Key, presignedUrl } = await urlResponse.json()
  onProgress(10)

  // ── Step 3: Upload directly to R2 ────────────────────────────────────
  onStageChange('uploading')

  await uploadWithProgress(file, presignedUrl, (percent) => {
    // Map 10% to 90% for the actual upload progress
    onProgress(10 + Math.floor(percent * 0.8))
  }, signal)

  onProgress(90)

  // ── Step 4: Confirm upload with Supabase ─────────────────────────────
  onStageChange('confirming')

  const confirmResponse = await fetch(`${EDGE_FUNCTION_BASE}/confirm-upload`, {
    method: 'POST',
    headers,
    signal,
    body: JSON.stringify({ fileId }),
  })

  if (!confirmResponse.ok) {
    const err = await confirmResponse.json()
    throw new Error(err.error || 'Failed to confirm upload.')
  }

  onProgress(100)
  onStageChange('complete')

  return {
    fileId,
    r2Key,
    originalName: file.name,
    mimeType: file.type,
    sizeBytes: file.size,
  }
}

// ── XMLHttpRequest upload with real progress ──────────────────────────────
// fetch() does not support upload progress. XHR does.

const uploadWithProgress = (file, presignedUrl, onProgress, signal) => {
  return new Promise((resolve, reject) => {
    const xhr = new XMLHttpRequest()

    xhr.upload.addEventListener('progress', (e) => {
      if (e.lengthComputable) {
        const percent = Math.round((e.loaded / e.total) * 100)
        onProgress(percent)
      }
    })

    xhr.addEventListener('load', () => {
      if (xhr.status === 200 || xhr.status === 204) {
        resolve()
      } else {
        reject(new Error(`Upload failed with status ${xhr.status}`))
      }
    })

    xhr.addEventListener('error', () => reject(new Error('Upload failed. Check your connection.')))
    xhr.addEventListener('abort', () => reject(new Error('Upload cancelled.')))

    // Support AbortController for cancellation
    signal?.addEventListener('abort', () => xhr.abort())

    xhr.open('PUT', presignedUrl)
    xhr.setRequestHeader('Content-Type', file.type || 'application/octet-stream')
    xhr.send(file)
  })
}

// ── Get download URL ──────────────────────────────────────────────────────

export const getDownloadUrl = async ({
  fileId,
  linkSlug,
  unlockType,
  sessionKey,
  forceDownload = true,
}) => {
  const headers = { 'Content-Type': 'application/json' }

  // Add auth header if user is logged in
  try {
    const authHeaders = await getAuthHeaders()
    Object.assign(headers, authHeaders)
  } catch {
    // Anonymous viewer — no auth header
  }

  const response = await fetch(`${EDGE_FUNCTION_BASE}/get-download-url`, {
    method: 'POST',
    headers,
    body: JSON.stringify({ fileId, linkSlug, unlockType, sessionKey, forceDownload }),
  })

  if (!response.ok) {
    const err = await response.json()
    throw new Error(err.error || 'Could not get download URL.')
  }

  const { downloadUrl, expiresIn } = await response.json()
  return { downloadUrl, expiresIn }
}

// ── Delete a file ─────────────────────────────────────────────────────────

export const deleteFile = async (fileId) => {
  const headers = await getAuthHeaders()

  const response = await fetch(`${EDGE_FUNCTION_BASE}/delete-file`, {
    method: 'POST',
    headers,
    body: JSON.stringify({ fileId }),
  })

  if (!response.ok) {
    const err = await response.json()
    throw new Error(err.error || 'Failed to delete file.')
  }

  return true
}
```

```javascript
// src/components/FileUploadZone.jsx
import { useState, useRef, useCallback } from 'react'
import { uploadFile, validateFile, formatFileSize, getFileEmoji } from '../services/uploadService'

const STAGE_LABELS = {
  validating:  'Checking file...',
  preparing:   'Preparing upload...',
  uploading:   'Uploading...',
  confirming:  'Finishing...',
  complete:    'Upload complete',
}

const FileUploadZone = ({
  bucket = 'content',
  onUploadComplete,   // ({ fileId, r2Key, originalName, mimeType, sizeBytes }) => void
  onUploadError,      // (errorMessage) => void
  onRemove,           // () => void
  accept,             // optional MIME type string e.g. "video/*"
  maxFiles = 1,
  label = 'Drag a file or tap to attach',
  sublabel,
  currentFile,        // { originalName, sizeBytes, mimeType } — for showing existing file
  disabled = false,
}) => {
  const [isDragging, setIsDragging] = useState(false)
  const [isUploading, setIsUploading] = useState(false)
  const [progress, setProgress] = useState(0)
  const [stage, setStage] = useState(null)
  const [error, setError] = useState(null)
  const inputRef = useRef(null)
  const abortRef = useRef(null)

  const handleFile = useCallback(async (file) => {
    setError(null)

    // Client-side validation
    const validation = validateFile(file, bucket)
    if (!validation.valid) {
      setError(validation.error)
      onUploadError?.(validation.error)
      return
    }

    setIsUploading(true)
    setProgress(0)
    setStage('preparing')

    // Create abort controller for cancellation
    abortRef.current = new AbortController()

    try {
      const result = await uploadFile(file, bucket, {
        onProgress: setProgress,
        onStageChange: setStage,
        signal: abortRef.current.signal,
      })
      onUploadComplete?.(result)
    } catch (err) {
      if (err.message !== 'Upload cancelled.') {
        setError(err.message)
        onUploadError?.(err.message)
      }
    } finally {
      setIsUploading(false)
      setStage(null)
      abortRef.current = null
    }
  }, [bucket, onUploadComplete, onUploadError])

  const handleDrop = useCallback((e) => {
    e.preventDefault()
    setIsDragging(false)
    const file = e.dataTransfer.files?.[0]
    if (file && !disabled) handleFile(file)
  }, [handleFile, disabled])

  const handleDragOver = (e) => {
    e.preventDefault()
    if (!disabled) setIsDragging(true)
  }

  const handleDragLeave = () => setIsDragging(false)

  const handleInputChange = (e) => {
    const file = e.target.files?.[0]
    if (file) handleFile(file)
    // Reset input so same file can be re-selected
    e.target.value = ''
  }

  const handleCancel = () => {
    abortRef.current?.abort()
    setIsUploading(false)
    setProgress(0)
    setStage(null)
  }

  const handleRemove = () => {
    setError(null)
    onRemove?.()
  }

  // ── Existing file state ──────────────────────────────────────────────
  if (currentFile && !isUploading) {
    return (
      <div style={{
        border: '1.5px solid #E8E8E8', borderRadius: '12px',
        padding: '14px 16px', background: 'white',
      }}>
        <div style={{ display: 'flex', alignItems: 'center', gap: '12px' }}>
          <span style={{ fontSize: '28px' }}>
            {getFileEmoji(currentFile.originalName, currentFile.mimeType)}
          </span>
          <div style={{ flex: 1, minWidth: 0 }}>
            <div style={{
              fontSize: '14px', fontWeight: 800, color: '#111',
              whiteSpace: 'nowrap', overflow: 'hidden', textOverflow: 'ellipsis',
            }}>
              {currentFile.originalName}
            </div>
            <div style={{ fontSize: '12px', color: '#AAAAAA', fontWeight: 600, marginTop: '2px' }}>
              {formatFileSize(currentFile.sizeBytes)}
            </div>
          </div>
          {!disabled && (
            <button
              onClick={handleRemove}
              style={{
                width: '32px', height: '32px', borderRadius: '50%',
                border: '1.5px solid #E8E8E8', background: 'white',
                cursor: 'pointer', fontSize: '14px', color: '#888',
                display: 'flex', alignItems: 'center', justifyContent: 'center',
                flexShrink: 0,
              }}
            >
              ✕
            </button>
          )}
        </div>
      </div>
    )
  }

  // ── Uploading state ──────────────────────────────────────────────────
  if (isUploading) {
    return (
      <div style={{
        border: '1.5px solid #E8E8E8', borderRadius: '12px',
        padding: '20px 16px', background: 'white',
      }}>
        <div style={{ display: 'flex', justifyContent: 'space-between', marginBottom: '10px' }}>
          <span style={{ fontSize: '13px', fontWeight: 700, color: '#333' }}>
            {STAGE_LABELS[stage] || 'Uploading...'}
          </span>
          <span style={{ fontSize: '13px', fontWeight: 800, color: '#E8312A' }}>
            {progress}%
          </span>
        </div>

        {/* Progress bar */}
        <div style={{
          height: '6px', background: '#F0F0F0', borderRadius: '3px', overflow: 'hidden',
        }}>
          <div style={{
            height: '100%', width: `${progress}%`,
            background: '#E8312A', borderRadius: '3px',
            transition: 'width 200ms ease',
          }} />
        </div>

        <button
          onClick={handleCancel}
          style={{
            marginTop: '12px', fontSize: '12px', fontWeight: 700,
            color: '#AAAAAA', background: 'none', border: 'none',
            cursor: 'pointer', padding: '0',
          }}
        >
          Cancel upload

  // ── Drop zone state ──────────────────────────────────────────────────
  return (
    <div>
      <div
        onDrop={handleDrop}
        onDragOver={handleDragOver}
        onDragLeave={handleDragLeave}
        onClick={() => !disabled && inputRef.current?.click()}

          borderRadius: '12px',
          padding: '24px 16px',

            {sublabel}
```

### Block 14 — Cleanup: Orphaned Placeholder Files

Placeholder file records are created in the DB before upload starts. If the user closes the browser mid-upload the R2 object never gets created but the DB record stays with `is_placeholder = true`. A cleanup function runs daily.

Add this SQL function to Supabase — run via SQL editor:

```sql
-- Deletes placeholder file records older than 2 hours (upload never completed)
CREATE OR REPLACE FUNCTION cleanup_orphaned_files()
RETURNS INTEGER AS $$
DECLARE
  deleted_count INTEGER;
BEGIN
  DELETE FROM public.files
  WHERE is_placeholder = true
    AND created_at < NOW() - INTERVAL '2 hours';

  GET DIAGNOSTICS deleted_count = ROW_COUNT;
  RETURN deleted_count;
END;
$$ LANGUAGE plpgsql SECURITY DEFINER;

-- Schedule it with pg_cron (available in Supabase Pro)
-- If on free tier, call this from a daily Edge Function instead
SELECT cron.schedule(
  'cleanup-orphaned-files',
  '0 3 * * *',  -- 3am daily
  'SELECT cleanup_orphaned_files();'
);
```

### What Parts 5 Onwards Connect To

Every part from here that involves files imports from `uploadService.js` and uses `FileUploadZone`. The pattern is consistent:

```javascript
// In any component that handles file upload:
import FileUploadZone from '../components/FileUploadZone'
import { deleteFile } from '../services/uploadService'

// Upload
<FileUploadZone
  bucket="content"
  onUploadComplete={(result) => {
    setFileId(result.fileId)
    setFileName(result.originalName)
  }}
  onUploadError={(msg) => showToast(msg, 'error')}
  onRemove={() => {
    if (fileId) deleteFile(fileId)
    setFileId(null)
    setFileName(null)
  }}
  label="Drag your file here or tap to browse"
  sublabel="PDF, video, template, image — up to 500MB"
/>
```

No mock file handling anywhere in the app after this part. Part 5 uses this service immediately — every link created in the dashboard writes a real file record and a real R2 object.
